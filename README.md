# ScriptSeekers Writings

A minimalist, mobile-first poetry and prose showcase built using pure HTML, CSS, and JavaScript.  
This project focuses on **reading experience over features** — no frameworks, no distractions, just words.

---

## What This Project Is

This is a **single-page literary portfolio** designed to present original writings in a calm, intentional way.

It is NOT:
- a blog engine
- a CMS
- a framework-based site

It IS:
- a clean poetry/prose showcase
- summary-first, reader-controlled content
- fully responsive and fast-loading

---

## Features

- Expandable **Read / Hide** cards for each writing
- Dark-first UI with automatic light-mode support
- Fully responsive (mobile, tablet, desktop)
- Poetry-friendly formatting (line breaks preserved)
- Zero dependencies (no libraries, no frameworks)
- Single-file simplicity

---

## Tech Stack

- **HTML5** – structure
- **CSS3** – styling, responsiveness, themes
- **Vanilla JavaScript** – content toggling & mobile UX

No build tools. No npm. No config files.
